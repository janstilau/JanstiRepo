## Array --> 标准意义的 c 风格

end 就是size 所指的元素的下一个位置的地址.

++ 就是指针的增长

## list 双向链表
头指针. 为了编程的方便.
iterator 不在是简单的指针, 而是一个对象, 里面有一个 node 值.
通过 iterator 进行插入
通过 iterator 进行删除
通过 iterator 进行遍历.
begin 就是头结点的 next
end 是尾巴结点
size 是维护的一个值
Iterator 里面, 包含一个 node.
++, 是操作这个 node 的指向. 
* 是 node.value.
