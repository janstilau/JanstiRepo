说明

  • 未作特殊说明，所有接口参数均做url编码
  • 出错代码赋值给返回字段c,200为接口成功,其他值客户端均认为相关业务功能失败

  • 接口返回值数据分为c,h,m,n,p 几个主要字段
    c为状态码字段
    n为弹框提示内容字段,n中有内容客户端自动弹框
    p为主要业务逻辑内容字段
    m,h为相关错误提示字段,作为调试阶段技术人员提示数据
    ts服务器当前时间戳 

    {
c = 200;
h = "";
m = "成功";
n = "";
p =     {
    str = "";
};
ts = 1481518556724;
}

  • 请求方法为POST

基础参数

参数名 参数定义 注释
api_version 接口版本号
client_version 客户端版本号
platform 平台 0-iphone1-android
device_id 设备id
time 当前时间戳
distributor 渠道名称
login_uid 登陆者uid 未登录时不需要
session_key 登陆者session 未登录时不需要
